BoomBigNose Template

About This Template

BoomBigNose is a comprehensive development environment that combines multiple powerful services for building modern applications. This template includes everything you need to get started with no-code development, workflow automation, object storage, text-to-speech, databases, and AI.

Included Services

n8n - Workflow Automation Tool

Fair-code licensed workflow automation tool that helps you connect various services and automate tasks.

NCA-Toolkit - No-Code Architects Toolkit

A toolkit for building applications without writing code.

MinIO - Object Storage Service

High-performance, S3-compatible object storage system.

Kokoro TTS - Text-to-Speech Service

Convert text to natural-sounding speech.

Supabase - Open Source Firebase Alternative

An open-source Firebase alternative with PostgreSQL database, authentication, storage, and more.

Redis - In-memory Data Store

In-memory data structure store, used as a database, cache, and message broker.

Ollama - Open Source LLM Runner

Run open-source large language models locally.
